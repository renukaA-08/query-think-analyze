# Easy SQL Problems

This repository contains my solutions to **Easy-level SQL problems** from **LeetCode** and **DataLemur**.

Each problem includes:

-  Problem statement
-  Problem link
-  My SQL solution
-  Step-by-step approach
-  SQL concepts used

The main goal of this repository is to improve SQL problem-solving skills .

## Repository Structure

Each problem folder contains:

```
Problem Name/
│── solution.sql
│── README.md
```

The `README.md` for each problem includes:

- Problem Description
- Problem Link
- Approach
- SQL Topics Used

## Platforms

- LeetCode SQL
- DataLemur SQL

## About the Solutions

The approaches in this repository are my own implementations and optimizations while solving the problems. 

## Purpose

This repository serves as my personal SQL practice journal . It documents my learning process, different SQL techniques, and optimized solutions to easy-level questions.

Feel free to explore the solutions and compare different approaches for solving the same problem.
